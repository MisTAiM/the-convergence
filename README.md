## ⚡ What Is This

> *"Large-scale systems do not fail randomly. They align, compress, and then transition."*

The Convergence is a **single-page civilizational intelligence dashboard** that aggregates live data from 11 government and financial APIs, cross-references it against 10 global news sources simultaneously, runs it through validated analytical frameworks (the Four Horsemen model, Fourth Turning cycle theory, Empire collapse patterns, Lippmann's pseudo-environment thesis), and synthesizes everything into a single composite score: the **Civilizational Stress Index**.

This is not a news aggregator. It is a **pattern detection engine for civilizational inflection points**.

## 🧠 Core Intelligence Modules

<details>
<summary><strong>📊 Civilizational Stress Index (CSI)</strong> — composite 0–100 score</summary>
<br>

The CSI synthesizes 8 signal domains into a single score using a weighted correlation model:

```
CSI = ( Σ ( wᵢ × sᵢ × vᵢ × cᵢ ) ) / N

wᵢ = signal weight     sᵢ = normalized value
vᵢ = volatility coeff  cᵢ = cross-domain correlation factor
```

Domains: **Climate · Geopolitical · Financial · Technological · Social · Resource · Institutional · Nuclear**

The score renders as a live SVG gauge with historical line chart. Updates every 20 minutes. The gauge needle and arc animate on each data cycle.

</details>

<details>
<summary><strong>⚔️ Four Horsemen Tracker</strong> — calibrated composite scoring</summary>
<br>

Revelation's Four Horsemen applied as a **systems dynamics model** — not metaphysics. Each horseman feeds the next through documented feedback loops. War → supply disruption → famine → immune stress → disease → deaths → more conflict.

| Horseman | Primary Data Sources | 2026 Score |
|----------|---------------------|------------|
| **WAR** | WB military spend trend + 8-source conflict headlines + GDACS alerts + named war zones | ~72/100 |
| **FAMINE** | WB extreme poverty % + under-5 mortality + food headlines + Brent Oil price | ~50/100 |
| **PLAGUE** | WHO RSS disease alerts + outbreak headlines + AMR resistance status | ~38/100 |
| **DEATH** | Life expectancy direction + horsemen confluence amplifier + displacement headlines | ~45/100 |

Hard caps: WAR 85 (100 = WWIII), FAMINE 80, PLAGUE 75, DEATH 75. Scoring was mathematically validated and calibrated against 2026 baseline inputs. See [math tests](#-horsemen-calibration) below.

</details>

## 🧪 Horsemen Calibration

The Four Horsemen scoring went through rigorous math testing after discovering the original formulas were broken:

```python
# ORIGINAL BROKEN SCORING (deployed before math tests):
WAR:    base 50 + headlines × 8          = 100/100 always (capped every cycle)
FAMINE: base 30 + poverty×80M + headlines = 67/100 (poverty M count formula wrong)
PLAGUE: base 20 + headlines × 15         = 67/100 with ZERO active pandemic
DEATH:  100 - (lifeExp - 50)             = 76/100 for 73.5yr (INVERTED — improving = worse)

# RECALIBRATED (math tested against 2026 real inputs):
WAR:    base 28 + calibrated addends, hard cap 85  = ~72/100 ✓
FAMINE: base 22 + calibrated multipliers            = ~50/100 ✓
PLAGUE: base 12 + WHO×5 + headlines×3 + AMR +4     = ~38/100 ✓
DEATH:  base 15 + confluence amplifier (not inverted) = ~45/100 ✓
```

The key insight for DEATH: life expectancy **improving** should produce a **lower** death score. A 73.5-year global average is historically good. The old formula gave it 76/100.

> Note: These are NOT the same value. Learned through a very specific bug where adding `position: relative; z-index: 1` to the nav group collapsed it out of its fixed stacking context, making the nav scroll with the page. The canvas must be at `-1`, not `0`.

## 🔌 Adding a New API Endpoint

Every endpoint follows the same pattern:

```javascript
// api/new-signal.js
const https = require('https');

function fetchData(url) {
  return new Promise((resolve, reject) => {
    https.get(url, { timeout: 9000, headers: { 'User-Agent': 'TheConvergence/3.0' } }, (res) => {
      let data = '';
      res.on('data', c => data += c);
      res.on('end', () => { try { resolve(JSON.parse(data)); } catch { resolve(null); } });
    }).on('error', reject).on('timeout', function() { this.destroy(); reject(new Error('timeout')); });
  });
}

module.exports = async (req, res) => {
  res.setHeader('Access-Control-Allow-Origin', '*');
  res.setHeader('Cache-Control', 'public, max-age=900'); // 15 min

  try {
    const data = await fetchData('https://your-source.gov/api/endpoint');
    res.json({ data, updated: new Date().toISOString() });
  } catch (err) {
    res.status(500).json({ error: err.message });
  }
};
```

Then wire into `js/app.js`:
```javascript
DataEngine.on('updated', async db => {
  try { renderNewModule(db); } catch(e) { console.error('new-module', e); }
});
```
